# X Post Analyzer

把 X（Twitter）推文数据 CSV 扔进来，自动分类对比——只摊数据，不做诊断。

## 两种用法

### 给 AI Agent 用（推荐）

把 `SKILL.md` 和 `analyze.py` 放到 AI Agent 的 skills 文件夹下，然后说：

> 分析我的 X 推文数据 /path/to/data.csv

AI 自动加载这个 skill，调用 analyze.py 跑完整分析。

### 命令行直接用

```bash
python3 analyze.py ~/Downloads/x_data.csv
```

## 功能

- **4 张基础对比表**：方向涨粉效率 / 回复帖真相 / 排行榜 / 方向分布
- **11 维 DNA 分析**：人称、数字、长度、开头风格、高频词、写作指纹、emoji、互动深度、时段、爆款连锁、句式
- **TOP5 最佳推文**：涨粉最多的 5 条原文展示
- **定位检测**（可选）：对比账号定位声明 vs 数据现实
- **列名自动识别**：中英文列名都能用，缺列自动跳过对应分析

## 原理

不做判断，只做对比。因为对比本身就够狠了。

回复帖 55%，涨粉 0 — 数字不会撒谎，结论你自己看。
